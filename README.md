# Smart Healthcare Assistant for Disease Prediction and Management

## Project Description
This project uses Machine Learning to predict diseases such as diabetes based on patient health parameters. The system analyzes input health data and provides predictions to assist healthcare decision making.

## Technologies Used
- Python
- Machine Learning
- TensorFlow / Keras
- Scikit-learn
- Pandas
- NumPy

## Files in Project
app.py – Main application file
healthcare_project.ipynb – Model training notebook
diabetes_model.keras – Trained model
scaler.pkl – Data scaler

## How to Run
1. Install requirements
2. Run app.py
